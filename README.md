# Git Assignment

## How I Will Use Git and VS Code
I will use Git to track changes to my code over time using commits.
Each commit is like a save point I can return to. I'll use branches
to work on new features without affecting the main codebase.

VS Code has built-in Git support (the Source Control panel on the left)
that lets me stage files, write commit messages, and see changes visually.

## What This Assignment Demonstrates
This assignment shows how GitHub works as a remote backup and collaboration
platform. It demonstrates branching (parallel development), merging
(combining work), and pull requests (proposing and reviewing changes
before they become permanent).
